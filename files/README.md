# AI교육컨설팅 — 홈페이지 (정적 프로토타입)

3인의 AI 강사가 함께 운영하는 교육 컨설팅 서비스의 홈페이지입니다.
벤치마크 사이트(kgpt.or.kr)의 구조적 힘을 참고하되, "협회"가 아닌 "3인 크리에이티브 컬렉티브" 톤으로 완전히 다른 포지셔닝을 취했습니다.

## 디자인 시스템

### 컬러
- **Base (배경)** — `#F5F0E6` Warm Ivory
- **Text** — `#1A1E1C` Deep Ink
- **Accent** — `#2D4A37` Deep Forest

### 강사별 톤 변주 (같은 계열 톤의 진하기 차이)
- **강사 A** — `#1A3525` (가장 진함) · 배경 `#C5D0BD`
- **강사 B** — `#2D4A37` (핵심 톤) · 배경 `#CED9C3`
- **강사 C** — `#4E6955` (가장 밝음) · 배경 `#D8E0CC`

### 타이포그래피
- **Headlines** — Noto Serif KR (에디토리얼 무드)
- **Body** — Pretendard (깔끔한 산세리프)
- Google Fonts와 jsDelivr CDN으로 로드됨

## 파일 구조

```
site/
├── index.html              메인 페이지
├── instructor-a.html       강사 A 프로필
```

## 주요 기능

### 메인 페이지 (index.html)
- 히어로 · Our Approach 밴드 · 3인 강사 카드 · 교육 사례 · 4갈래 프로그램 · CTA · 푸터

### 강사 프로필 페이지 (instructor-a/b/c.html)
- 대형 포트레이트 · 시그니처 인용구 · 주제 태그 · 에세이 · 강의 가능 주제 · 대표 사례
- 하단에 "Next Instructor" 링크로 3인을 자연스럽게 순회

### 강의 의뢰 (inquiry.html)
- **Step 1 · 강의 희망일** — 캘린더에서 최대 3개 날짜를 클릭하면 1→2→3 순위로 시그니처 컬러로 표시
- **Step 2 · 의뢰 정보** — 기관·교육 내용·담당 강사·자유 메모 입력
- **Step 3 · 확인** — 섹션별 수정 링크, 접수 확인 즉시 연락 안내
- **Step 4 · 완료** — 3-dot 마크로 감사 인사
- localStorage에 날짜 선택 임시 저장 (새로고침 후에도 유지)

### 3인 전용 대시보드 (dashboard.html)
- **비밀번호 모달** — 별도 로그인 페이지 없이 모달로 접근 제어 (데모 비밀번호: `trio2026`)
- **robots noindex** 메타 태그로 검색엔진 노출 차단
- 의뢰 통계 · 리스트 (상태별 필터) · 주제·지역 분포

## 실서비스 전환 시 필요한 작업

1. **로고 교체** — 현재는 3-dot 마크로 대체되어 있음. 업로드하신 워드마크 로고를 SVG 또는 PNG로 저장 후 `logo.svg` 파일명으로 붙이고 HTML의 `.logo` 영역을 `<img>`로 교체
2. **강사 포트레이트** — 현재는 실루엣 SVG. 통일된 무드의 실제 사진 촬영 후 교체
3. **강사 정보 확정** — 이름, 사업체명, 저서 등 실제 정보로 업데이트
4. **교육 사례** — 실제 진행 사례의 썸네일 이미지 + 본문 교체
5. **백엔드 연결** — inquiry 폼 데이터를 받을 서버 엔드포인트 구현 (Supabase, Firebase, Vercel Functions 등 권장)
6. **이메일 발송** — 접수 시 의뢰자 자동 응답 메일 + 3인에게 알림 메일 (Resend, EmailJS 등)
7. **대시보드 인증** — 현재는 클라이언트 사이드 데모 비밀번호. 실서비스에선 서버 사이드 인증으로 전환
8. **대시보드 실제 데이터** — 의뢰 폼 데이터베이스와 연결

## 기술 스택 제안 (실서비스 전환 시)

- **프론트엔드** — Next.js 14 (App Router) + TailwindCSS로 리팩토링
- **DB + 인증** — Supabase (의뢰 저장 + 대시보드 인증을 함께 처리)
- **이메일** — Resend
- **배포** — Vercel
- **도메인** — 확정 후 연결
